# [atom-jsfmt](https://atom.io/packages/jsfmt)

Format your JavaScript and JSON using [jsfmt](https://github.com/rdio/jsfmt).

## Install

```bash
$ apm install jsfmt
```

## Usage

- Packages > jsfmt > Format JavaScript
- Right Click > Format JavaScript

## Config

### Format on Save

Format JavaScript on Save.
